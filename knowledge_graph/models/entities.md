# Entidades del Knowledge Graph

| Entidad | Descripción |
|---------|-------------|
| Institution | Organización productora o custodial |
| Source | Plataforma o sistema de datos |
| Dataset | Conjunto de datos identificable |
| Variable | Medida o atributo temático |
| Service | Servicio de acceso (OGC, REST, …) |
| API | Interfaz programática |
| UseCase | Caso de uso / problema territorial |
| Reference | Citación / DOI / documentación |
| Country | Ámbito nacional |
| Theme | Dominio temático |
| BibliographicIndex | Índice de literatura científica (Scholar, Scopus, WoS) |
